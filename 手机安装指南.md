# 📱 手机安装指南 - 马上金融调解计算器

## ✅ 您的工具已经是PWA（渐进式Web应用）

**不需要打包成APK**，直接可以在手机上"安装"成App使用！

---

## 📦 安装步骤（3分钟完成）

### 方法一：Android手机（Chrome/Edge浏览器）

1. **在手机浏览器中打开工具**
   - 如果文件在电脑上：先上传到网络（可以用GitHub Pages、Netlify等免费托管）
   - 或者：在电脑上运行简单的HTTP服务器，手机在同一WiFi下访问

2. **添加到主屏幕**
   - 打开工具后，点击浏览器右上角 **⋮**（菜单按钮）
   - 选择 **"安装应用"** 或 **"添加到主屏幕"**
   - 确认安装

3. **完成！**
   - 手机主屏幕会出现App图标
   - 点击图标，像原生App一样全屏运行
   - 离线也能用（部分功能）

---

### 方法二：iPhone/iPad（Safari浏览器）

1. **在Safari中打开工具**
   - 同样需要先托管到网络，或用电脑搭建本地服务器

2. **添加到主屏幕**
   - 点击浏览器底部的 **分享按钮**（方框带向上箭头）
   - 滑动找到 **"添加到主屏幕"**
   - 点击 **"添加"**

3. **完成！**
   - 主屏幕出现App图标
   - 点击即可使用

---

## 🚀 快速测试：用电脑搭建本地服务器

如果文件在电脑上，可以用简单方法让手机访问：

### Windows电脑：

1. 打开命令提示符（CMD）
2. 进入工具目录：
   ```cmd
   cd E:\collector_assistant
   ```
3. 启动简单HTTP服务器：
   ```cmd
   python -m http.server 8000
   ```
   （如果没有Python，可以用 `npx serve` 或 `php -S localhost:8000`）

4. 查看电脑IP地址：
   ```cmd
   ipconfig
   ```
   （找到IPv4地址，例如：`192.168.1.100`）

5. 手机在同一WiFi下访问：
   ```
   http://192.168.1.100:8000
   ```

6. 按照上面的步骤安装到主屏幕

## 📲 推荐：免费托管到网络（永久解决）

### 方案1：GitHub Pages（推荐）
1. 在GitHub创建仓库
2. 上传所有文件到仓库
3. 开启GitHub Pages
4. 获得免费网址：`https://您的用户名.github.io/仓库名`
5. 手机访问这个网址，安装到主屏幕

### 方案2：Netlify Drop（最简单）
1. 打开 https://app.netlify.com/drop
2. 拖拽 `E:\collector_assistant` 文件夹到网页
3. 自动部署，获得免费网址
4. 手机访问，安装到主屏幕
